Acesse Leilões para Android – política de privacidade e permissões solicitadas

O aplicativo Acesse Leilões para Android não coleta nenhum dado pessoal.

O aplicativo Acesse Leilões para Android solicita permissões de internet para fazer download dos dados dos leilões.

O aplicativo Acesse Leilões para Android solicita permissões de armazenamento para salvar os editais e os comunicados dos leilões.
